# 严格表格逐项迭代

## 何时使用

当用户给出以下任一种输入，并明确或隐含要求“按表格一个个做方向迭代”时，必须优先使用本技能：

- Excel
- CSV
- 表格截图
- 从表格提取出的文本
- 飞书/文档里的方向表
- 一行一个方向的清单

尤其当用户强调以下要求时，更要严格使用本技能：

- “一个个处理”
- “不要漏项”
- “严格按我给的表格来”
- “不要自己合并总结”
- “每一行都要扩”
- “按顺序逐条迭代”

## 核心目标

把用户提供的表格视为**待逐项执行的任务清单**，而不是灵感参考。你的职责不是先做抽象总结再挑一部分展开，而是：

1. 完整识别表格中的每一行或每一个方向项
2. 按原顺序逐项处理
3. 为每一项分别做方向迭代
4. 保证输出能清楚对应回原表格中的那一项

## 硬性执行规则

以下规则必须同时满足：

### 1. 不跳项

只要该行是有效方向项，就必须处理。

- 不要因为你觉得某些行相似就跳过
- 不要因为某些行信息少就忽略
- 不要只挑你认为“更好展开”的几项来做

如果某一行信息确实不足，也要明确标记该行信息不足，并在可推断范围内继续输出该行的迭代结果，而不是省略。

### 2. 不合并

除非用户明确要求“把相近行合并处理”，否则：

- 不要把多行合并成一个方向
- 不要把多个原方向抽象成一个大类后只输出一个结果
- 不要用“这些都差不多”来替代逐项处理

### 3. 保持原顺序

输出顺序必须与表格中的原始顺序一致。

如果原表顺序是第1行、第2行、第3行……，你的输出也必须按这个顺序展开。不要自行重排为“你认为更合理”的顺序。

### 4. 一项对应一项

每个输出单元都必须能回指到原表格中的具体行。

每一项输出都至少要包含：

- 原始行号或原始序号
- 原方向名称或该行识别出的核心标识
- 该行对应的新方向迭代结果

不允许输出一大段无法对应原表行的混合内容。

### 5. 先识别表，再迭代

在正式扩展前，先把你识别到的表格项列出来，确认你理解的处理对象是什么。

如果表格很大，可以先用紧凑格式列出：

- 第1行：xxx
- 第2行：xxx
- 第3行：xxx

然后再逐项进入扩展。

## 行识别规则

把“可独立展开方向”的最小单位视为一行任务。优先识别这些字段：

- 方向名
- 标签名
- 主题词
- 对应参考图
- 卖点描述
- 场景描述
- 备注

如果表格列名不规范，也要根据内容判断哪一行代表一个独立方向。

如果同一行里有多个字段，默认把它们一起作为该行的上下文，而不是拆散到别的行里处理。

## 逐项迭代流程

对每一行都按以下顺序执行：

### 第一步：定位该行

先写清楚你正在处理哪一行。

例如：

- 第3行｜极寒据点防御
- 第7行｜废墟资源争夺

### 第二步：提炼该行已有方向

说明这一行原本在表达什么，至少说清：

- 题材焦点
- 视觉抓手
- 可延展空间
- 容易重复的风险

### 第三步：做该行的方向迭代

基于这一行，产出新的方向迭代，不要跳到别的行上去借题发挥。

如果用户没有另行指定数量，默认每一行至少给：

- 3 个新方向；或
- 1 个新方向 + 多组提示词

具体数量可根据用户当前任务要求调整，但要确保每一行都有独立产出。

### 第四步：保持与其他行区隔

当前行的迭代结果必须与其他行清楚分开。不要把不同原行的扩展结果混在同一个小节里。

## 输出格式

优先使用严格结构化输出，推荐两种格式：

### 格式A：逐项分块

适合行数不太多时。

每一项结构至少包含：

- 原表序号
- 原方向
- 原方向简析
- 新方向迭代
- 提示词

### 格式B：严格对应表

适合用户本来就在看表格、且希望直接逐行核对时。

建议列：

- 原表序号
- 原方向名称
- 原方向理解
- 迭代方向1
- 迭代方向2
- 迭代方向3
- 备注/去重提醒

无论使用哪种格式，都必须满足“原表一项对应输出一项”。

## 禁止行为

以下行为视为错误：

- 只总结共性，不逐项展开
- 把十几行方向合并成几个大类后不回到每一行
- 只处理你认为重要的几行
- 输出与原表顺序不一致
- 省略无法确定的行而不说明
- 用泛化总结替代逐项迭代

## 例外处理

只有在以下情况，才可以暂停严格逐项迭代：

1. 用户明确说“先挑最值得做的几项”
2. 用户明确说“先聚类再展开”
3. 表格本身严重损坏，导致无法判断哪些内容属于独立行

如果是第3种情况，也要先明确告诉用户哪些行无法可靠识别，而不是擅自忽略。

## 质量标准

最终结果必须让用户能直接核对：

- 我的表格有没有被逐项处理
- 有没有漏掉某一行
- 每一行有没有被单独迭代
- 输出顺序是不是和原表一致

如果用户能一眼对照原表逐项检查，你就达到了本技能的目标。
