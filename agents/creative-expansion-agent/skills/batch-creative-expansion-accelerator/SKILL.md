# 批量创意扩展加速器

## 何时使用

当用户出现以下需求时，优先使用这个技能：

- 给了多张参考图、拼图、表格、Excel、截图或方向列表，希望快速拆解共性与差异。
- 希望基于现有方向继续扩展，但明确要求避免重复、避免同质化、提高出稿效率。
- 希望把少量参考方向拆成可批量生产的系列化方向，例如扩到几十组或上百组素材。
- 希望直接拿到结构化结果，例如分析表、方向表、批量中文出图提示词表。

如果用户只是要一两条灵感，不需要批量处理，也没有参考素材，则不必强行套用本技能的完整流程，可只借用其中的“去重检查”和“方向拆分”规则。

## 目标

把“参考素材理解 → 共性抽取 → 去重避坑 → 系列化拆分 → 新方向扩展 → 中文提示词生成”压缩成一套稳定流程，减少重复分析、减少空泛创意、减少方向撞车，并让输出更适合批量出图。

## 输入识别

先判断用户给的是哪一类输入：

1. **参考图型**：单张图、多张图、拼图、截图、文档内嵌图。
2. **表格型**：Excel、CSV、表格截图、方向清单、标签表。
3. **混合型**：既有参考图也有表格/方向说明/卖点说明。
4. **轻输入型**：只有少量关键词或一句方向要求。

然后提取四类关键约束：

- 世界观或题材边界
- 必须保留的核心卖点
- 明确禁止重复的内容
- 目标规模（例如 5 个方向、20 组素材、100 组扩展）

如果目标规模没有说清楚，默认先输出“高质量方向拆分方案 + 每个方向少量样例提示词”，而不是直接无上限铺开。

## 核心流程

按下面顺序执行，不要跳步。

### 第一步：快速建立参考索引

如果参考项很多，先给每个参考项编号或命名，再继续分析。

- 单图：直接按主视觉命名
- 多图：按“参考图1、参考图2……”或按画面主题命名
- 表格：按每一行的方向名或标签名建立索引

目标不是复述原内容，而是为后续“逐项分析”和“去重比较”建立稳定锚点。

### 第二步：逐项分析“为什么成立”

对每个参考项，至少从以下维度中挑出最关键的 4 到 6 个进行分析：

- 主题表达
- 主体设计
- 主体关系
- 场景机制
- 构图方式
- 镜头距离与视角
- 光线与色彩
- 材质细节
- 情绪氛围
- 广告传播性
- 系列化潜力

不要只写“氛围好”“画面强”这种空话。要说清楚：

- 视觉抓手到底是什么
- 它靠什么形成记忆点
- 哪些元素值得复用
- 哪些元素再重复会立刻撞车

### 第三步：抽取共性、变量和重复风险

把所有参考项汇总成三层结构：

1. **必须保留的共性**
   - 例如世界观基调、材质审美、冷暖关系、商业海报感
2. **可变化的变量**
   - 例如主体数量、场景类型、叙事焦点、镜头、动作、卖点承载方式
3. **高重复风险项**
   - 例如总是同一种构图、同一种角色站位、同一种“废墟+风雪”背景模板

这一阶段的目标是建立“保留什么、变化什么、避开什么”的扩展边界。

### 第四步：优先做系列化拆分

当用户要求大量扩展时，不要直接连续生成很多相似新方向，而要先把方向拆成若干系列母题。优先从这些维度拆：

- 生存机制
- 资源争夺
- 交通与迁移
- 角色群像关系
- 末日遗迹
- 极寒设施/据点
- 科技改装
- 英雄化表达
- 孤独求生表达
- 奢侈感或高端质感表达
- 静态海报感
- 动态冲突感

拆分原则：

- 每个系列必须有清晰的广告表达重点。
- 每个系列之间必须在主体关系、叙事重心或视觉抓手上明显不同。
- 不接受只换地名或只换场景名词但画面结构不变的伪扩展。

### 第五步：生成新方向

每个新方向都必须包含：

- **新方向名称**：短、清晰、能区分
- **方向描述**：一句话说明相对原参考的新变化
- **差异点**：至少明确指出该方向与已有方向的关键区别

新方向必须建立在已有参考分析之上，属于“方向迭代”，不是完全脱离参考的自由发散。

优先补足以下空白：

- 参考中尚未覆盖的题材角度
- 尚未覆盖的情绪和叙事关系
- 更适合系列化批量扩展的视觉模块
- 更适合广告传播爆点的主视觉机制

### 第六步：生成中文出图提示词

为每个新方向生成多组中文提示词时，遵循以下规则：

- 每组提示词都要能独立形成一张完整广告画面。
- 不要写成关键词堆砌；要写成有明确画面组织的完整提示词。
- 每组提示词都尽量包含：主体、场景环境、构图、镜头、光线、色彩、材质、情绪、3D渲染质感。
- 同一方向下的多组提示词要体现“同母题下的镜头/构图/情绪变体”，而不是简单换词。

如果用户没有要求具体数量，默认每个新方向给 3 到 5 组高质量提示词；如果明确要求 5 组或更多，则按要求输出。

## 输出模板

默认优先用结构化表格或分块清单输出，顺序如下：

### A. 参考分析表

每个参考项至少包含：

- 参考项
- 这张图为什么成立
- 可复用元素
- 高重复风险点

### B. 批量扩展策略表

当用户要批量扩展时，输出：

- 拆分维度
- 系列方向
- 适合扩展的原因
- 建议批量生成方式

### C. 新方向扩展表

每个新方向至少包含：

- 参考来源
- 新方向名称
- 方向描述
- 差异点
- 提示词1
- 提示词2
- 提示词3
- 提示词4（如需要）
- 提示词5（如需要）

如果用户只要其中一部分，就裁剪输出；但如果用户给了大量参考素材，默认不要只给结论，要保留“分析依据”。

## 去重规则

生成前必须做一次自检，避免以下问题：

- 多个新方向只是换了场景名词，本质构图和叙事完全一样
- 多组提示词只是同一句话的近义改写
- 画面仍旧过度依赖单一视觉母题
- 新方向与参考图的重复度过高，看不出新增视角
- 只强调美术风格，没有广告传播重点

如果发现撞车，优先改以下变量，而不是只改形容词：

- 主体关系
- 叙事焦点
- 镜头视角
- 前中后景组织
- 光线机制
- 卖点承载方式
- 动作和张力来源

## 质量标准

最终结果必须满足：

- 用户能快速看懂每个参考项为什么有效
- 用户能一眼看出新方向之间的差异
- 用户能直接拿提示词去出图，而不是还要二次补写
- 批量扩展逻辑清晰，适合继续从 5 个方向扩到更多素材
- 保持原世界观一致，同时确实拉开变化

## 两个典型请求形态

### 形态1：用户给了表格和多张参考图

用户可能会说：

“我给你一份现有方向表和几张已经跑出来的图，帮我继续扩更多方向，别重复。”

你应该：

1. 先按参考项建立索引。
2. 逐项分析图和表里的已有方向为什么成立。
3. 总结共性与重复风险。
4. 先做系列化拆分。
5. 再输出新的方向和对应中文提示词。

### 形态2：用户只给了少量关键词，但想批量扩展

用户可能会说：

“围绕极寒、生存、废墟、未来科技，帮我扩成一批广告图方向。”

你应该：

1. 先补出隐含的世界观骨架。
2. 给出建议的系列拆分维度。
3. 输出多组差异明确的新方向。
4. 每个方向附上可直接出图的中文提示词。

## 最后原则

这个技能的重点不是让输出更长，而是让分析更快、扩展更稳、方向更不重复、结果更适合批量生产。
