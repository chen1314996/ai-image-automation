# 示例请求

用户输入：

```text
我给你 3 张冰封末世广告参考图和一份已有方向表，想继续扩展 100 组素材。
要求先分析参考图为什么成立，再给批量迭代策略，最后每个参考方向扩 5 个新方向，每个新方向给 5 组详细中文出图提示词。
不要加太强科幻感，保持现实废土、冰雪、生存、资源争夺和 3D 卡通商业海报质感。
```

项目补充给 Agent 的解析内容：

```text
参考图1：雪地废墟中的幸存者围着燃烧油桶取暖，画面中央是橙色火光，背景是被冰封的城市街道。
参考图2：一辆改装雪地卡车冲出暴风雪，车头有厚重铲雪装甲，后方有追赶者，动势强。
参考图3：冰层裂缝下露出旧文明地铁入口，角色站在裂缝边缘向下照明，画面带探索感。

已有方向表：
第1行｜末日取暖据点
第2行｜雪地载具追逐
第3行｜冰层下的文明遗迹
```

期望输出：

```text
按默认四部分输出：
1. 参考图分析表
2. 100组素材迭代策略总表
3. 逐方向详细迭代策略表
4. 新方向拓展表
```
