# Windows 一键启动

这个文件用于说明如何在 Windows 上一键启动自动化生图平台。

## 双击启动

请双击同目录下的：

```text
Windows_start.bat
```

双击后会自动打开一个新的命令行窗口，并执行这些步骤：

1. 进入当前项目文件夹。
2. 检查 `Node.js` 和 `npm` 是否可用。
3. 如果缺少 `node_modules` 依赖，会自动执行 `npm install`。
4. 启动本地服务器：`node server.js`。
5. 等服务器可访问后，自动打开平台页面：

```text
http://localhost:3055/
```

## 停止服务器

服务器运行时，请保留启动窗口不要关闭。

需要停止时，在启动窗口里按：

```text
Ctrl+C
```

或者直接关闭该命令行窗口。

## 端口已被占用

如果 `3055` 端口已经有服务器在运行，脚本不会重复启动服务器，会直接打开平台页面。

如果页面打不开，请先关闭旧的服务器进程，再重新双击 `Windows_start.bat`。

## 注意

Markdown 文件 `.md` 本身不能作为 Windows 启动脚本执行。真正用于双击启动的是同目录下的 `Windows_start.bat`。
