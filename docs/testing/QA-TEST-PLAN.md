# Web应用程序质量检查(QA)测试流程文档

## 测试环境要求

- 操作系统: Windows 11
- Node.js: v18+ 
- 浏览器: Chromium (通过Playwright)
- 运行模式: 带界面模式 (headless: false)

## 启动应用

```bash
cd "C:\Users\dd\Desktop\项目"
npm start
```

访问: http://localhost:3066
