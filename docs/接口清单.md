# 接口清单

这份文档按功能列出后台接口，方便你知道页面按钮大概会调用哪里。

## 浏览器和文件夹

| 方法 | 地址 | 说明 |
| --- | --- | --- |
| `GET` | `/` | 打开主页面 |
| `POST` | `/api/count-images` | 统计文件夹里的图片数量 |
| `POST` | `/api/select-folder` | 打开本机文件夹选择窗口 |
| `POST` | `/api/open-website` | 打开单个网站 |
| `POST` | `/api/open-both-websites` | 打开自动化需要的网站 |
| `POST` | `/api/close-browser` | 关闭浏览器 |
| `GET` | `/api/browser-status` | 获取浏览器状态 |

| `GET` | `/api/config/legil-generation` | 获取 Legil 生成参数 |
| `POST` | `/api/config/legil-generation` | 保存 Legil 生成参数 |
| `GET` | `/api/config/notifications` | 获取通知配置 |
| `POST` | `/api/config/notifications` | 保存通知配置 |
| `GET` | `/api/config/workflow` | 获取完整工作流配置 |
| `POST` | `/api/config/workflow` | 保存完整工作流配置 |
| `GET` | `/api/config/resize` | 获取批量改尺寸配置 |
| `POST` | `/api/config/resize` | 保存批量改尺寸配置 |
| `GET` | `/api/config/creative` | 获取创意拓展配置 |
| `POST` | `/api/config/creative` | 保存创意拓展配置 |
| `GET` | `/api/config/legil-ref-folder` | 获取 Legil 参考图文件夹 |
| `POST` | `/api/config/legil-ref-folder` | 保存 Legil 参考图文件夹 |

## 豆包

| 方法 | 地址 | 说明 |
| --- | --- | --- |
| `POST` | `/api/doubao/full-automation` | 根据参考图生成提示词 |
| `GET` | `/api/doubao/extracted-prompts` | 获取最近一次豆包提示词 |
| `POST` | `/api/doubao/upload-and-prompt` | 兼容旧流程的豆包上传和提示词接口 |

## Legil

| 方法 | 地址 | 说明 |
| --- | --- | --- |
| `POST` | `/api/legil/generate` | 用单条提示词生成图片 |
| `POST` | `/api/legil/batch-generate` | 批量生成图片 |
| `GET` | `/api/legil/task-status` | 查看当前 Legil 任务状态 |
| `POST` | `/api/legil/stop` | 停止当前 Legil 任务 |
| `POST` | `/api/legil/resize-batch` | 批量改尺寸 |
| `GET` | `/api/legil/creative-resume` | 查看创意拓展可继续任务 |
| `GET` | `/api/legil/creative-progress` | 查看创意拓展进度 |
| `POST` | `/api/legil/creative-resume/clear` | 清除创意拓展继续任务 |
| `POST` | `/api/legil/creative-batch` | 执行创意拓展批量生成 |

## 完整工作流

| 方法 | 地址 | 说明 |
| --- | --- | --- |
| `POST` | `/api/workflow/start` | 开始完整工作流 |
| `GET` | `/api/workflow/status` | 查看完整工作流状态 |
| `GET` | `/api/workflow/resume-info` | 查看是否有可继续任务 |
| `POST` | `/api/workflow/resume` | 继续完整工作流 |
| `POST` | `/api/workflow/clear-resume` | 清除完整工作流继续状态 |
| `GET` | `/api/workflow/extracted-prompts` | 获取工作流最近提取到的提示词 |
| `POST` | `/api/workflow/stop` | 停止完整工作流 |
