## 详细变更记录

### 1. public/index.html

#### 变更 1：创意拓展停止后操作入口
**新增/调整：**
- 继续任务按钮。
- 新任务按钮。
- 停止后展示可恢复操作。
- 继续任务默认深灰色，停止后才可点击。

#### 变更 2：创意拓展运行模式
**新增：**
- 有头模式。
- 无头模式。

**说明：**
有头模式保留原工作流，无头模式用于后台运行，减少浏览器窗口干扰。

#### 变更 3：Agent 对话框入口
**新增：**
- 打开 Agent 对话框按钮。
- 上传表格/图片/文件夹入口。
- 文字指令输入区。
- 输出表格下载与提示词提取入口。

---

### 2. server.js

#### 变更 1：创意拓展恢复状态
**新增/增强：**
- 创意拓展任务停止后保存恢复信息。
- 后端提供恢复信息查询。
- 启动新任务时可区分普通启动与继续任务。

#### 变更 2：Agent 后端接口
**新增能力：**
- 接收前端上传资料和文字指令。
- 后台调用 Agent。
- 返回任务状态和结果表格。

#### 变更 3：安全配置
**处理：**
- API Key 由后端读取。
- 不在前端页面回显。
- 错误信息脱敏。

---

### 3. creative-agent-service.js / creative-agent-worker.js

#### 变更：Agent 后台运行
**目标：**
Agent 创意拓展不影响 Legil 图片生成和正在进行的工作流。

**实现：**
- 通过 Worker 执行 Agent。
- 主线程只维护任务状态。
- 前端轮询任务状态。

---

### 4. creative-table-parser.js

#### 变更：表格提示词解析
**目标：**
将 Agent 输出表格当作“本地表格”使用。

**处理：**
- 解析 Excel/CSV。
- 提取方向、标题、提示词列。
- 为后续提示词预览和批量生成提供结构化数据。

---

### 5. creative-agent-quality.js

#### 变更：质量评估
**目标：**
解决项目 Agent 输出详细程度不足的问题。

**初步指标：**
- 方向数量。
- 每个方向是否有 5 组提示词。
- 提示词长度和细节密度。
- 是否缺字段。
