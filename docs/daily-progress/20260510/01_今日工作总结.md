### 4. 改尺寸与量产参数隔离
**问题：**
改尺寸页面设置 Legil 参数时，不能影响量产页面。

**解决方案：**
- 量产使用 `config.legilGeneration`
- 改尺寸使用 `config.resizeLegilGeneration`
- 后端改尺寸配置存入 `resize.generationSettings`
- 改尺寸任务运行时临时传入参数，结束后恢复 Legil 自动化实例原参数

**验证：**
- 保存改尺寸参数不会改变量产参数
- UI 点击改尺寸参数按钮不会影响量产页面按钮状态

### 5. 修复 Legil 横版/竖版生成状态检测
**问题现象：**
- 方版图片能识别生成完成
- 横版、竖版图片在黑底容器中显示，系统一直等待

**根因：**
旧逻辑只判断 `img` 元素自身宽高和 `naturalWidth/naturalHeight`，横版高度较小、竖版宽度较小，会被阈值过滤。

**修复方案：**
- 从只识别 `img` 升级为识别输出槽位/卡槽
- 横版、竖版、方版统一按 Legil 输出结果容器判断
- 保存阶段也使用同一套输出定位逻辑

**验证：**
- Playwright 模拟横版 16:9、竖版 9:16、方版 1:1 均检测成功
- `waitForGenerationComplete()` 能稳定返回完成

### 6. 优化 Legil 大图保存
**问题：**
同一批生成图中，有些能保存大图，有些因 HTTP 403 只能截图。

**改进方案：**
- 从弹窗中收集更多图片来源：`currentSrc`、`src`、`data-src`、`data-original`、`data-url`、`srcset`、父级下载链接
- 优先构造去除缩略参数后的原图地址
- 直链失败后增加浏览器真实打开图片响应获取 bytes
- 再失败尝试页面内 `fetch`
- 最后才用大图弹窗截图兜底

---

### 7. 恢复 Legil 登录态持久化
**问题：**
关闭 Legil 浏览器窗口后重新打开又需要登录。

**根因：**
混用了 `storage_state.json` 和 `browser_data`，旧登录态可能覆盖新登录态。

**解决方案：**
- 取消每次启动导入 `storage_state.json`
- 只使用 `browser_data` 持久化用户目录
- 恢复旧版可用的 `browser_data_backup_20260508_154344`
- 加入浏览器关闭/断开监听，关闭窗口后后端状态自动重置
