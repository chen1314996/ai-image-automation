## 今日核心成果
- 接通飞书企业自建应用机器人，完成 SDK 长连接、消息指令、卡片按钮后台回调。
- 优化飞书控制卡片，保留高频按钮：状态、进度、开始量产、继续任务、停止任务、重启服务。

- 新增 Windows 计划任务 `AI Image Automation Watchdog`，并改为隐藏启动，避免周期性弹出终端窗口。
- 量产、改尺寸、创意拓展同步支持有头模式/无头模式，默认尽量使用无头模式。
- 修复飞书继续任务找不到创意拓展可恢复任务的问题。
- 调整飞书进度信息，使 Legil 提示词进度、成功/失败/保存数量尽量与自动化平台一致。
- 优化 Legil 保存图片命名，去掉“提示词1”这类泛化命名，并为量产结果保留简短提示词标题。
- 将量产页面、改尺寸页面的 Legil 检测和保存链路对齐创意拓展页面，避免同一类 Legil 输出在不同页面表现不一致。
