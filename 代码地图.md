# 代码地图

这份文件也是给你看的，放在根目录，方便你快速找代码。

## 一句话总览

```text
前端按钮 -> public/js/*.js -> src/routes/*.routes.js -> 后台业务文件 -> 自动化平台
```

比如“开始完整工作流”：

```text
public/js/workflow.js
  -> src/routes/workflow.routes.js
  -> workflow-controller.js
  -> doubao-automation.js
  -> src/services/legil/
```

## 根目录保留的核心文件

| 文件 | 作用 |
| --- | --- |
| `server.js` | 项目启动兼容入口，保持 `node server.js` 可用 |
| `package.json` | 项目信息和 npm 命令 |
| `automation-config.json` | 页面和后台保存的运行配置 |
| `automation-secrets.json` | 密钥配置，不要随便改 |
| `doubao-automation.js` | 豆包/火山方舟提示词生成 |
| `workflow-controller.js` | 完整工作流总调度 |
| `playwright-controller.js` | 浏览器自动化控制 |
| `logger.js` | 实时日志和前端 SSE 推送 |
| `feishu-*.js` | 飞书通知、控制、CLI 桥接相关 |

## 后台启动

| 文件 | 作用 |
| --- | --- |
| `src/server/start.js` | 后台启动入口，只负责串联 |
| `src/server/context.js` | 后台共用配置、状态、工具函数 |
| `src/server/routes.js` | 统一挂载所有接口模块 |
| `src/server/runtime-services.js` | 飞书 CLI、守护进程、健康监控启动 |
| `src/server/shutdown.js` | 服务关闭清理 |
| `src/server/startup-banner.js` | 控制台启动提示 |

## 后台接口

| 文件 | 作用 |
| --- | --- |
| `src/routes/browser.routes.js` | 浏览器打开/关闭、文件夹选择、图片统计 |
| `src/routes/config.routes.js` | 豆包、Legil、通知、工作流、改尺寸、创意配置 |
| `src/routes/doubao.routes.js` | 豆包提示词生成和最近提示词读取 |
| `src/routes/legil.routes.js` | Legil 单图、批量、停止、改尺寸、创意批量 |
| `src/routes/workflow.routes.js` | 完整工作流启动、停止、续跑、状态 |
| `src/routes/creative-agent.routes.js` | 创意 Agent 表格解析、运行、查询、下载 |
| `src/routes/feishu.routes.js` | 飞书通知、飞书事件、飞书 CLI 控制 |
| `src/routes/health.routes.js` | 健康检查、通知状态、守护进程状态 |
| `src/routes/logs.routes.js` | 实时日志 SSE 和最近日志 |

## Legil 模块

| 文件/目录 | 作用 |
| --- | --- |
| `legil-automation.js` | 根目录兼容入口，旧代码继续可用 |
| `src/services/legil/index.js` | 创建 LegilAutomation 实例 |
| `src/services/legil/config-methods.js` | 文件夹、参数、文件名、参考图扫描 |
| `src/services/legil/page-actions.js` | 打开页面、上传参考图、输入提示词 |
| `src/services/legil/generation-settings.js` | 设置模型、比例、分辨率、数量并点击生成 |
| `src/services/legil/generation-flow.js` | 单次生成主流程 |
| `src/services/legil/output-detection.js` | 结果识别入口 |
| `src/services/legil/output/` | 等待生成、扫描结果、失败识别 |
| `src/services/legil/image-save.js` | 图片保存入口 |
| `src/services/legil/save/` | 下载、截图兜底、保存主流程 |

## 常用命令

```bash
npm start
npm run test:creative-agent
npm run test:feishu
npm run test:quick
```

这次重构保持 `npm start` 和 `node server.js` 不变。
