## 详细变更记录

### 1. doubao-automation.js

#### 变更1: 替换豆包网页自动化为 API 调用
**删除/废弃能力:**
- 打开豆包网页
- 上传图片到豆包网页
- 等待网页回复
- 从 DOM 中抓取豆包回复
- 每张参考图新建豆包网页对话

**新增能力:**
- 读取本地参考图
- 调用豆包图文大模型 API
- 使用固定提示词模板请求 5 组生图提示词
- 解析 API 返回内容为提示词数组

---

#### 变更2: API 配置预留
**配置项:**
- API Key
- 模型 ID
- API Base URL
- 固定指令模板

**安全说明:**
真实 API Key 不记录在文档中，不写入 Git 跟踪文件。

---

### 2. workflow-controller.js

#### 变更1: 工作流接入 API 模式
**新流程:**
```text
本地参考图 → 豆包 API → 5 组提示词 → Legil 循环生成 → 保存输出
```

**保留逻辑:**
- 输入图片排序
- 输出命名
- Legil 参考图上传
- Legil 生成等待
- 图片保存
- 工作流停止控制

---

### 3. server.js

#### 变更1: 豆包配置与安全配置接口
**修改内容:**
- 增加豆包 API 配置读取/保存
- 敏感信息写入本地忽略文件
- 前端只需要填写 API Key 和模型 ID

---

#### 变更2: 新增改尺寸批处理接口
**新增/增强接口:**
- `GET /api/config/resize`
- `POST /api/config/resize`
- `POST /api/legil/resize-batch`
- `GET /api/legil/task-status`
- `POST /api/legil/stop`

**运行逻辑:**
- 扫描改尺寸输入文件夹
- 按名称自然排序
- 每张图上传到 Legil
- 使用固定提示词和独立生成参数
- 输出到改尺寸输出目录

---

#### 变更3: 服务器优雅关闭
**修改内容:**
- 增加 `SIGINT` / `SIGTERM` 处理
- 按 `Ctrl+C` 时关闭浏览器并退出服务

---

#### 变更4: 撤销打开后强制模型切换
**修改原因:**
打开 Legil 后二次检测/切换模型导致页面打开变慢。

**最终策略:**
- 打开 Legil 只打开网页
- 默认模型由 Legil 页面和浏览器 profile 记忆
- 生成前再按具体任务参数应用模型

---

### 4. legil-automation.js

#### 变更1: 支持直接参考图路径
**修改内容:**
- `uploadReferenceImage()` 支持 `options.referenceImagePath`
- 改尺寸任务可逐张上传输入目录图片

---

#### 变更2: 横版/竖版输出检测
**修改内容:**
- 增加输出槽位识别
- 放宽对 `img` 自身宽高的方图阈值依赖
- 支持从 `data-src`、`srcset` 等字段提取输出地址

**验证:**
Playwright 模拟横版、竖版、方版输出均检测成功。

---

#### 变更3: 大图保存增强
**保存优先级:**
1. 去除缩略参数后的原图地址
2. Playwright request 下载
3. 浏览器真实打开图片响应
4. 页面内 fetch
5. blob/data 图片
6. 大图弹窗截图

---

#### 变更4: 弹窗关闭与生成前清理
**修改内容:**
- 每轮生成前检查并关闭未关闭的大图弹窗
- 避免弹窗遮挡参数按钮或生成按钮

---
